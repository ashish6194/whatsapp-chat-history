# Voice dictation

> Use push-to-talk voice input with Claude Code.

## Enable

Run `/voice` to toggle voice dictation. Requires a Claude.ai account.

## Usage

Hold Space to talk. Release to transcribe. Text inserts at cursor.

## Supported languages

20+ languages including English, Spanish, French, German, Japanese, Chinese, and more.

## Rebind push-to-talk key

Configure in keybindings.json under `voice:pushToTalk`.
