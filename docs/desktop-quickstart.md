# Get started with the desktop app

> Install Claude Code on desktop and start your first coding session.

The desktop app gives you Claude Code with a graphical interface: visual diff review, live app preview, GitHub PR monitoring, parallel sessions, scheduled tasks, and remote execution.

## Install

1. Download Claude for macOS or Windows
2. Sign in with your Anthropic account
3. Open the Code tab

The desktop app includes Claude Code. No need to install Node.js or the CLI separately.

## Start your first session

1. **Choose environment and folder**: Select Local, Remote, or SSH. Pick your project directory.
2. **Choose a model**: Select from the dropdown next to the send button.
3. **Tell Claude what to do**: Type your task and press Enter.
4. **Review and accept changes**: See diff view, accept or reject each change.

## What's next

- **Interrupt and steer**: Click stop or type corrections mid-action
- **Give more context**: Use @filename, attach images/PDFs, drag and drop files
- **Use skills**: Type `/` to browse commands and workflows
- **Review changes**: Click diff indicator to review file by file
- **Adjust control**: Switch permission modes (Ask, Auto accept, Plan)
- **Add plugins**: Browse and install from the plugin manager
- **Preview your app**: Run dev server in embedded browser
- **Track PRs**: Monitor CI status with auto-fix and auto-merge
- **Schedule tasks**: Set up recurring automated tasks
- **Scale up**: Open parallel sessions, send work to cloud

## Coming from the CLI?

Desktop runs the same engine with a graphical interface. Both share configuration (CLAUDE.md, MCP servers, hooks, skills, settings).
