# Claude Code overview

> Claude Code is an agentic coding tool that reads your codebase, edits files, runs commands, and integrates with your development tools.

## Get started

Available in Terminal, VS Code, JetBrains, Desktop app, and Web.

### Install (Terminal)

```bash
curl -fsSL https://claude.ai/install.sh | bash
cd your-project && claude
```

## What you can do

- Automate tedious tasks (tests, lint, migrations)
- Build features and fix bugs
- Create commits and pull requests
- Connect tools with MCP
- Customize with CLAUDE.md, skills, and hooks
- Run agent teams and custom agents
- Pipe, script, and automate with CLI

## Use Claude Code everywhere

| I want to... | Best option |
|---|---|
| Continue session from phone | Remote Control |
| Push events from chat | Channels |
| Automate PR reviews | GitHub Actions / GitLab CI/CD |
| Get automatic code review | Code Review |

| Debug web apps | Chrome |
