# Continue local sessions from any device with Remote Control

> Continue a local Claude Code session from your phone, tablet, or browser.

Remote Control connects claude.ai/code or the Claude mobile app to a session running on your machine. Everything runs locally.

## Start a session

```bash
claude remote-control          # Server mode
claude --remote-control        # Interactive with remote enabled
/remote-control                # From existing session
```

## Connect from another device

Open the session URL, scan QR code, or find session in claude.ai/code.

## Requirements

- Pro, Max, Team, or Enterprise plan
- Claude Code v2.1.51+
- Team/Enterprise: admin must enable Remote Control
