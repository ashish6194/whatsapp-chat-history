# Troubleshooting

> Solutions for common Claude Code issues.

## Authentication issues

- Run `/logout` then `/login`
- Check subscription status
- Verify network connectivity

## Permission errors during installation

- Don't use `sudo npm install -g`
- Fix npm permissions or use native installer

## Search and discovery issues

- Install ripgrep if search fails
- Check file permissions

## Performance issues

- Use `/clear` between tasks
- Reduce MCP server count
- Adjust effort level

## Get more help

Run `claude doctor` for installation diagnostics.
