# Tools reference

> Complete list of tools available to Claude Code.

## Built-in tools

| Tool | Category | Description |
|------|----------|-------------|
| Read | File | Read file contents |
| Edit | File | Edit existing files |
| Write | File | Create or overwrite files |
| Glob | Search | Find files by pattern |
| Grep | Search | Search file contents |
| Bash | Execution | Run shell commands |
| WebFetch | Web | Fetch web content |
| WebSearch | Web | Search the web |
| AskUserQuestion | Interaction | Ask user for input |
| Agent | Orchestration | Spawn subagents |
| LSP | Code intelligence | Language server operations |

## MCP tools

Tools from connected MCP servers follow format: `mcp__<server>__<tool>`
