# VS Code extension

> Use Claude Code in Visual Studio Code with inline diffs, @-mentions, and conversation history.

## Install

Search "Claude Code" in Extensions (Cmd+Shift+X) or install from:
- [VS Code Marketplace](vscode:extension/anthropic.claude-code)
- [Cursor](cursor:extension/anthropic.claude-code)

## Features

- Inline diff viewing
- @-mention files and selections
- Plan review and approval
- Conversation history
- Browser automation with Chrome extension

## Get started

Open Command Palette (Cmd+Shift+P), type "Claude Code", select "Open in New Tab".

## Security and privacy

- Extension runs locally
- Code sent to Anthropic API for processing
- Same data policies as CLI
