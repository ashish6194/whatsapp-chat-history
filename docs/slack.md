# Claude Code in Slack

> Mention @Claude in Slack to start Claude Code tasks from team conversations.

Mention @Claude in a Slack channel or thread to start a web session. Claude analyzes the conversation context and works on your code.

## Setup

1. Run `/install-slack-app` in Claude Code
2. Connect your GitHub repositories
3. Mention @Claude in any channel

## How it works

- @Claude mention spawns a web session on claude.ai/code
- Claude reads the Slack thread for context
- Results posted back to the thread

## Requirements

- Teams or Enterprise plan
- GitHub app installed
- Slack app installed
