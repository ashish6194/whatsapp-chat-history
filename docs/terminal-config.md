# Terminal configuration

> Configure your terminal for optimal Claude Code experience.

## Shift+Enter support

Run `/terminal-setup` to install keybindings for terminals that need it (VS Code, Alacritty, Warp).

Works out of the box in iTerm2, WezTerm, Ghostty, and Kitty.

## Option as Meta key

Required for Alt shortcuts on macOS:
- **iTerm2**: Settings > Profiles > Keys > set Option to "Esc+"
- **Terminal.app**: Settings > Profiles > Keyboard > "Use Option as Meta Key"
