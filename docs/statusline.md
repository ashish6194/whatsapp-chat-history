# Status line

> Configure Claude Code's status line to display model, context usage, costs, and custom information.

## Configure

Run `/statusline` to set up, or describe what you want displayed.

## Available data

- `context_window.used_percentage`: context usage
- `model`: current model name
- `cost.total_usd`: session cost
- `session.duration`: session duration
- Custom shell commands

## Format

Configure in settings with the `statusLine` key.
